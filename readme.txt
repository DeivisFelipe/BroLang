A Linguagem BroLang, se as seguintes regras:

- A linguagem é case sensitive, ou seja, diferencia maiúsculas de minúsculas.
- A linguagem é composta por 3 tipos de dados: inteiro, real e char (uma cadeia de caracteres dentro de aspas duplas).
- se (condicao) { comando } senao { comando }
- enquanto (condicao) { comando }
- O valor real é representado por um ponto (.) e não por vírgula (,).
- A atribuicao é feita com o operador =.
- A linguagem possui operadores lógicos (e, ou, não) e operadores relacionais (igual, diferente, maior, menor, maior ou igual, menor ou igual). 
    && (e), || (ou), ! (não), == (igual), != (diferente), > (maior), < (menor), >= (maior ou igual), <= (menor ou igual)
- A linguagem possui operadores aritméticos (soma, subtração, multiplicação, divisão).
    + (soma), - (subtração), * (multiplicação), / (divisão)
- A linguagem aceita numeros negativos.
- A linguagem aceita comentários de bloco (/* */).
- Operações entre reais e inteiros resultam em reais.
- O código tem que começar com nulo principal () { }.
